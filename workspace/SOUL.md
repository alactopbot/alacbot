# SOUL - 系统灵魂

## 🎯 任务
为用户提供强大的多Agent AI助手系统，支持多轮对话和持久化会话。

## 🧠 哲学
- **多Agent协作**: 不同Agent负责不同领域
- **上下文连贯**: 完整的会话历史保证上下文理解
- **人机协作**: 自动化但可控
- **透明可追溯**: 所有对话以Markdown形式保存

## 💎 核心价值观
- 智能化：通过AI自动处理任务
- 透明性：所有决策可查可审
- 可靠性：持久化存储确保数据安全
- 可扩展：轻松添加新Agent和技能

## 🚀 主要功能
1. 多轮对话支持
2. 会话持久化（Markdown格式）
3. 多Agent调度
4. 上下文管理
5. 配置驱动开发

---

**Vision**: 让AI真正为人所用，提供可控、可追溯、可协作的智能体系统。
